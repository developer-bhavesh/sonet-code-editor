# SoneT 

## Modern Code Editor for Developers

SoneT is a cross-platform, lightweight code editor that combines native performance built with Go using Wails framework

###  **Project Goals**
- Provide a fast, responsive coding environment
- Minimize resource usage while maximizing functionality  
- Deliver intuitive user experience for developers
- Support multiple programming languages out-of-the-box


###  **Key Achievements**
- **Performance**: Sub-second startup time
- **Memory Efficient**: <100MB RAM usage
- **Cross-Platform**: Native builds for Linux, macOS, and Windows
- **Developer Experience**: Several keyboard shortcuts, command palette
- **File Management**: Advanced file explorer with search capabilities

###  **Project Stats**
- **Languages Supported**: 20+
- **Bundle Size**: <50MB
- **Startup Time**: <1 second
- **Development Time**: 2 months
- **Lines of Code**: ~2,500

###  **Impact**
SoneT addresses the gap between heavyweight IDEs and basic text editors, providing developers with a tool that's powerful enough for serious development work yet lightweight enough for quick edits and scripting.

**Status**: Beta Release Ready
**License**: MIT (Open Source)
**Platform**: Desktop (Linux, Windows,Mac OS)


## System Requirements

- **Linux**: Ubuntu 18.04+ or equivalent
- **macOS**: macOS 10.13+ (High Sierra)
- **Windows**: Windows 10+ (64-bit)
- **Memory**: 512MB RAM minimum
- **Storage**: 100MB available space

## Installation

Download the latest release from [GitHub Releases](https://github.com/developer-bhavesh/sonet-code-editor/releases/latest)

### Linux
1. Download `SoneT-linux-amd64`
2. Run the executable

### Windows
1. Download `SoneT-windows-amd64.exe`
2. Run the executable

### macOS (Coming Soon)
1. Download `SoneT-darwin-universal.dmg`
2. Open the DMG and drag to Applications
